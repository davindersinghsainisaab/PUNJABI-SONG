Computer Hardware Components

1. CPU (Central Processing Unit)

The CPU is the brain of the computer. It performs calculations and processes instructions.

2. RAM (Random Access Memory)

RAM stores temporary data that the computer is currently using.

3. Hard Disk / SSD

This hardware is used to store data permanently such as files, software, and operating systems.

4. Motherboard

The motherboard connects all hardware components of the computer together.

5. Input Devices

Examples include keyboard, mouse, scanner, and microphone.

6. Output Devices

Examples include monitor, printer, and speakers.
